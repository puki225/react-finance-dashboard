# Workflow

- After pushing commits to the designated feature branch and opening/updating a pull
  request, merge it immediately — do not ask for confirmation before merging.
- After making changes that require manual follow-up (running an n8n workflow,
  executing SQL directly in Postgres, triggering a sync endpoint, redeploying, etc.),
  always end the response with a clear "Next steps" list spelling out exactly what
  needs to be run and where.
